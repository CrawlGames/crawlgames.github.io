🎛️ Operational Control Scheme

Left / Right Arrow Keys (A/D): Slide the active block across your grid paths

Up Arrow Key (W / Space): Rotate the block shape 90 degrees clockwise

Down Arrow Key (S): Accelerate the block drop speed for faster row build-up

Shift / C Key: Store your current block shape in reserve for a tactical trade

Escape: Open system configuration options and active audio settings
